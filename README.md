# costco-special-offer
A simple Discord bot for Costco special offer notifications

![screenshot](assets/screenshot.png)

## Install
```
yarn install
```

## Set .env
```
APP_ID=
DISCORD_TOKEN=
PUBLIC_KEY=
CHANNEL_ID=
```

## Add cron
```
yarn pm2
```

## Run onetime (for every products. not only today's.)
```
yarn start:onetime
```
